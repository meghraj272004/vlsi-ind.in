# VLSI IND Clone

A professional semiconductor and VLSI design services platform built with React, Vite, Tailwind CSS, and Express.

## Features
- **Interactive Particle Background**: Circuit-themed particle system using `tsparticles`.
- **Animated Gradients**: SaaS-style 3D wave gradients for an immersive tech feel.
- **Modern UI/UX**: Glassmorphism, smooth animations with `framer-motion`, and responsive design.
- **Full-Stack Architecture**: Express server with Vite middleware integration.

## Tech Stack
- **Frontend**: React 18, Vite, Tailwind CSS, Motion, Lucide React.
- **Backgrounds**: @tsparticles/react, @tsparticles/slim.
- **Backend**: Express.js, tsx.

## Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
